## 💻 Sobre o projeto <a name="sobre"></a>

O projeto **Componente de Tabs com React** tem o objetivo de mostrar como fazer um componente de Tabs com React.

### 📥 Instalação <a name="instalacao"></a>

Clone o repositório

```sh
git clone https://github.com/SwannyAlves/tab-component-with-react
```

Instale as dependências rodando o comando

```sh
yarn
```

Digite o comando para ligar o servidor

```JS
yarn start
```
